## Packages
socket.io-client | Real-time multiplayer synchronization
framer-motion | Smooth animations, page transitions, and UI micro-interactions
lucide-react | High-quality icons for UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
- Using Socket.IO for real-time multiplayer state synchronization.
- Connecting to the default path ('/') for Socket.IO.
- Audio files are assumed to be referenced as generic paths; the audio hook gracefully handles missing files to prevent crashes.
- The app requires a modern browser for backdrop-filter (glassmorphism) support.
